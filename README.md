Сайт присвячений подорожам до Японії. Він пропонує інформацію про туристичні маршрути, визначні місця, місцеву культуру, традиції, їжу, а також розрахунок бюджету подорожі. Сайт спрямований на аудиторію, яка планує або цікавиться поїздками до Японії, надаючи їй всю необхідну інформацію для організації подорожі.

Запуск проєкту:
1.Клонувати репозиторій: git clone https://github.com/risewing/IC-33_appWEB-DomozhilkinOleg-FIOT-2025.git
2.Відкрити файл index.html у будь-якому сучасному браузері
або скористатися розширенням Live Server у Visual Studio Code.

Сайт надає користувачам інформацію про Японію як туристичний напрямок, знайомить з культурними особливостями країни, популярними місцями для відвідування та загальною атмосферою подорожей. Проєкт орієнтований на простоту використання, привабливий дизайн та зручну навігацію.
